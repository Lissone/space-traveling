## Description

Blog for news and tips for developers. Developed as a challenge to establish knowledge with CMS integration, and NextJs fundamentals.


## Requirements

* [Yarn](https://yarnpkg.com/)
* [Nodejs](https://nodejs.org/en/)

## Technologies

* NextJs
* Typescript
* Sass
* Prismic CMS
* Jest
* ESLint
* Prettier

## Usage

You can use this project as a template for another one, or clone it on your pc using the command:

```bash
git clone https://github.com/Lissone/space-traveling.git
cd space-traveling
```

Install dependencies using:
```bash
yarn
#or
npm run
```

Need to add environment variables in both projects:

```bash
# .\.env.local
PRISMIC_API_ENDPOINT=
PRISMIC_ACCESS_TOKEN=

```

Run application:
```bash
yarn dev
#or
npm run dev
```
